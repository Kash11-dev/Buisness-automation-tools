# Bill Category Finder

A Streamlit-based dashboard that allows users to upload sales Excel files and search for bills by category using partial keywords.

## Features

- Upload Excel files (.xls/.xlsx)
- Partial category search (e.g., "bio" → "BIONOVA")
- Multiple category search
- Bill-wise combined categories
- Interactive dashboard built with Streamlit

## Tech Stack

- Python
- Streamlit
- Pandas
- OpenPyXL
